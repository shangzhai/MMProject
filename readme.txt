To compile the program:
1. Simply click the Solution file "MMProject.sln" to launch this project in Visual Studio
2. Install OpenCV and configure the enviroment variables
3. Add Additional Library Directories to the project
4. Add the name of all modules which you want to use in OpenCV
5. Build the Solution and then start it.

To execute the program:
1. Launch the "MMProject.exe" under the "Release" folder
2. Select the mode, Image Retrieval or Object detection
3. Select your preferred image for Image Retrieval or Object Detection
4. The data will be shown in the text field
5. If you selected Image Retrieval, the results will be output to the folder "search_results". If you selected
Object Detection, the results will be output to the folder "detection_results"
6. Click OK or Cancel to exit
